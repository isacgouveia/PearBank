# Pear Bank

by Isac Gouveia

This is an educational project developed with the main goal of providing a practical environment for learning and aplying my knowledges. The project simulates the functionalities of a bank, offering different access' levels and operations for clients and managers.

## Features

### For Clients
- Customer Dashboard: Viewing balance and account information
- Performing Transfers: Sending and Receiving
- View Statements: Access a detailed history of all transactions (inflows and outflows).

### For Managers
- Management Dashboard: Overview of bank activities, such as number of clients and account activity.
- View Registered Users: Listing of all clients registered on the platform.
